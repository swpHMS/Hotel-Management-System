# Hotel Management System

## Introduction
**Hotel Management System** is a project for the course **SWP391 â€“ Software Development Project**.  
The system is developed to replace the current manual and fragmented processes in hotel management, including room reservations, guest service management, billing, and overall hotel operations.

The project aims to digitalize hotel workflows, reduce human errors, improve management efficiency, and enhance user experience.

---

## Project Objectives
- Manage room information and room availability
- Support room booking and guest management
- Manage hotel services and billing
- Standardize hotel operation processes
- Apply web development and database knowledge in a real-world system

---

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript  
- **Backend**: Java Servlet  
- **Database**: SQL Server  
- **Source Code Management**: Git, GitHub
